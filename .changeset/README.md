# Changesets

This folder is managed by `@changesets/cli` to help version and publish this package.

## Adding a changeset

Run `npx changeset` in the root of this repository. You'll be prompted to select the type of version bump (major, minor, or patch) and a summary of the change.

The changeset file will be created in this directory and should be committed along with your changes.

See the [changesets documentation](https://github.com/changesets/changesets/blob/main/docs/adding-a-changeset.md) for more details.
